# Coding Ninjas Invoice Generator Web App

A web application to generate PDF invoices from CSV data for the Alumni Connect Program.

## Features
- Upload CSV
- Add custom message and invoice month
- Auto-generate individual PDFs per mentor
- Download all as a ZIP

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy
You can deploy this app on Streamlit Cloud, Hugging Face Spaces, or Render.
